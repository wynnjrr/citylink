# CityLink Luxury Coaches

A full-stack modern website and booking portal for CityLink Luxury Coaches.

## Run

```powershell
npm.cmd start
```

Then open `http://localhost:3000`.

## Included

- Modern responsive frontend with motion, glass effects, animated route map, booking portal, routes, fares, loyalty, gallery, and contact sections.
- Node backend using built-in modules only.
- API endpoints for routes, bookings, and contact messages.
- Local JSON storage in `data/db.json`.

## API

- `GET /api/routes`
- `POST /api/bookings`
- `GET /api/bookings`
- `POST /api/contact`
